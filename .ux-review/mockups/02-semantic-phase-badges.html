Phase Badge Color System Comparison

BEFORE: Phase Badges (Current)

Spec Phases: Draft Ready InProgress Review Validating Done
Problem: InProgress, Review, and Validating are all identical black badges. Draft and Done are both identical gray. Users cannot distinguish phases at a glance. On the Flow Board, the only way to tell a card's phase is its column position.

AFTER: Semantic Phase Badges

Spec Phases: Draft Ready In Progress Review Validating Done
Product Status: Active Discovery Maintenance Sunset
Complexity: Low Medium High
Improvements: (1) Each phase has a unique color making workflow state instantly scannable. (2) Colors follow natural metaphor: gray=idle, blue=queued, amber=active, purple=needs attention, orange=testing, green=complete. (3) Complexity uses dashed-border style to visually separate it from phase badges. (4) All combinations pass WCAG AA contrast.
